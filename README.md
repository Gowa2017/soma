# \<Soma\>

Soma is the area builder for the Ranvier game engine

## Running

```
npm install
bower install
npm run start
```

## Creating Releases

To create distributables for Soma simply run `npm run package-<platform>` where platform is one of `linux`, `win`, or
`mac`, e.g., `npm run package-linux`
